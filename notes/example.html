    How to create and genereate this Couscous nonsense

    Directions

    • cd to dir where .md files lie i.e. Notes
    • run rm -r .couscous/
    • run /Users/brandonkupczyk/.composer/vendor/couscous/couscous/bin/couscous generate
    • cd .couscous
    • run cp -r generated/* ../
    • add to git and push!
